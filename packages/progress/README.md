# sfc-progress
> A sfc component

## Installation
```shell
npm i sfc-progress -D
```

## Usage
```javascript
import Vue from 'vue'
import SfcProgress from 'sfc-progress'

Vue.use(SfcProgress)
```

or

```javascript
import Vue from 'vue'
import { SfcProgress } from 'sfc-progress'

Vue.component('sfc-progress', SfcProgress)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
