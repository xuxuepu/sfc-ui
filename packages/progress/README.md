# sfc-progress
> A mint component

## Installation
```shell
npm i sfc-progress -D
```

## Usage
```javascript
import Vue from 'vue'
import MintProgress from 'sfc-progress'

Vue.use(MintProgress)
```

or

```javascript
import Vue from 'vue'
import { MintProgress } from 'sfc-progress'

Vue.component('sfc-progress', MintProgress)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
