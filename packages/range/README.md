# sfc-range
> A sfc component

## Installation
```shell
npm i sfc-range -D
```

## Usage
```javascript
import Vue from 'vue'
import SfcRange from 'sfc-range'

Vue.use(SfcRange)
```

or

```javascript
import Vue from 'vue'
import { SfcRange } from 'sfc-range'

Vue.component('sfc-range', SfcRange)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
