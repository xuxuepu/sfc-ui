# sfc-range
> A mint component

## Installation
```shell
npm i sfc-range -D
```

## Usage
```javascript
import Vue from 'vue'
import MintRange from 'sfc-range'

Vue.use(MintRange)
```

or

```javascript
import Vue from 'vue'
import { MintRange } from 'sfc-range'

Vue.component('sfc-range', MintRange)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
